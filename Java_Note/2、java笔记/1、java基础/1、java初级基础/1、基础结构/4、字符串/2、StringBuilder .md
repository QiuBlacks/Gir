# StringBuilder 

## 常用方法

```java
append
capacity() :返回当前容量。 
charAt(int index) 
deleteCharAt(int index) :删除 char在这个序列中的指定位置
getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin) :字符从该序列复制到目标字符数组 dst 。 
indexOf(String str, int fromIndex) :返回指定子串的第一次出现的字符串中的索引，从指定的索引开始
length() 
reverse() 
toString()
```
