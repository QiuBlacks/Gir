# **继承**

一、

## **1、特点：**

- 通过extends关键字,实现了继承之后,让类与类之间产生了关系。
- 继承的本质就是抽取共性代码,多个重复代码的向上抽取,简化代码。
- 类是对一批对象的抽象,继承是对一批类的抽象。
- 父类又被称为超类,或者基类,子类又被称为派生类。
- Java是面向对象语言,万事万物皆是对象,为了满足这个设计原则,所有的类都直接或者间接继承自Object类。

## **2、优缺点：**

**优点**

- - 简化了代码
  - 提高了扩展性
  - 提高了可维护性

**缺点：**

造成了耦合性过高,牵一发动全身

## **3、注意**

**Java之中只允许多层继承，不允许多重继承，Java存在单继承局限。**
