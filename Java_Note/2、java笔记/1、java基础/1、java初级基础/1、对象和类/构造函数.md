# 构造函数

## 一、构造函数

```
public class Math {
    public static void main(String[] args) {
        Person person;
        person = new Person("张三");
        person = new Person("李四");
    }

    public static class Person {
        private String name;
        Person(String name) {
            this.name = name;
        }
    }
}
```
