# 一、基本介绍
#### &emsp;&emsp;Java运算符大致分为:
- 逻辑运算符:  && &emsp; | |  &emsp;  ！
- 算术运算符:  +&emsp;  -&emsp;  *&emsp;  /&emsp;  +=&emsp;
- 位运算符: ^(异或)&emsp; |(或)&emsp; &（与） ~（取反）&emsp; >> &emsp;  <<  &emsp; <<< (无符号右移)
- 赋值运算符：= &emsp;+= &emsp;>>= &emsp;&= &emsp;/=
- 关系运算符：==&emsp; ！=&emsp; >&emsp; <&emsp; >=&emsp; <=
- 其他运算符: 条件运算符；instanceOf 运算符
