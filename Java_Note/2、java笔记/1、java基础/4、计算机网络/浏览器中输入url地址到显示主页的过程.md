# **浏览器中输入url地址到显示主页的过程**

## **一、过程**

1.浏览器查找域名的IP地址

2.浏览器向web服务器发送一个HTTP请求

3.服务器处理请求

4.服务器返回一个HTML响应

5.浏览器解析渲染HTML页面



## **二、使用的协议：**

DNS（域名系统）：获取域名对应IP。

TCP：与服务器建立TCP连接。

IP：建立TCP连接时，需要发送数据，发送数据在网络层使用IP协议。

OSPF（开放式最短路径优先）：IP数据包在路由器之间，路由选择使用OSPF协议。

ARP：路由器在于服务器通讯时，需要将IP地址转换为MAC地址，需要使用ARP协议。

HTTP：在TCP建立完成后，使用HTTP协议访问网页。
