# **指定运行级别**

 

## **一、运行级别说明：**

 

0 ：关机

 

1 ：单用户【找回丢失密码】

 

2：多用户状态没有网络服务

 

3：多用户状态有网络服务

 

4：系统未使用保留给用户

 

5：图形界面

 

6：系统重启

 

常用运行级别是 3和 5 ，要修改默认的运行级别可改文件

 

/etc/inittab的 id:5:initdefault:这一行中的数字
