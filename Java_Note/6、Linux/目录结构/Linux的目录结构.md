# Linux的目录结构

 

## 一、基本介绍

linux的文件系统是采用级层式的树状目录结构，在此结构中的最上层是根目录“/”，然后在此目录下再创建其他的目录。

深刻理解 linux树状文件目录是非常重要的，这里我给大家说明一下。

记住一句经典的话：==**在 Linux世界里，一切皆文件。**==
