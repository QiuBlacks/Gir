## 三、找出没有重复的数

适用于只有一个数出现一次，其他出现2次或者偶数次

```java
int find(int[] arr){
    int tmp = arr[0];
    for(int i = 1;i < arr.length; i++){
        tmp = tmp ^ arr[i];
    }
    return tmp;
}
```



## 四、2的n次方-->任意数的n次方

```java
int pow(int n){
    int sum = 1;
    int tmp = 2; //初始化为2^0001
    while(n != 0){
        if(n & 1 == 1){ //判断最后一位数是否为1
            sum *= tmp;  //相乘
        }
        tmp *= tmp; //2，4，8，16 对应1 1 1 1
        n = n >> 1; //右移一位
    }

    return sum;
}
```

2^1101 = 2^0001 * 2^0100 * 2^1000 = 2^1 *   2^4   *  2^8



## 五、找出不大于N的最大的2的幂指数

思路：将该数n的右边都变成0，然后右移一位

```java
int findN(int n){
    n |= n >> 1;//或运算
    n |= n >> 2;
    n |= n >> 4;
    n |= n >> 8 // 整型一般是 32 位，上面我是假设 8 位。
    return (n + 1) >> 1;
}
```
