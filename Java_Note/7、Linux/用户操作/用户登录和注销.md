#  **用户登录和注销**

 

## 1、基本介绍

 

1) 登录时尽量少用 root帐号登录，因为它是系统管理员，最大的权限，避免操作失误。可以利用普通用户登录，登录后再用”su - 用户名’命令来切换成系统管理员身份.

 

2) 在提示符下输入 logout 即可注销用户
