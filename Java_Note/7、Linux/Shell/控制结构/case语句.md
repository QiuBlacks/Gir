#  **case语句**

 

## **一、基本语法**

 

case $变量名 in

 

"值 1"）

 

如果变量的值等于值 1，则执行程序 1

；；

"值 2"）

如果变量的值等于值 2，则执行程序 2 

；；

…省略其他分支…

*）

；；

esac

如果变量的值都不是以上的值，则执行此程序
